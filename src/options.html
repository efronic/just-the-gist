Just The Gist — Options

Your API key is stored in Chrome sync storage. It never leaves your browser except when calling Google Gemini.

Higher detail levels include more transcript/page coverage, structured sections (key points, detailed walkthrough, quotes, data, Q&A), and longer output.

Transcripts

If enabled, summaries will rely only on visible page text and metadata. Useful for privacy or if transcript fetching causes issues.

Uncheck to hide the transcript UI (you can still fetch & use transcripts for summarization).

Tip: You can create an API key from the Google AI Studio.
